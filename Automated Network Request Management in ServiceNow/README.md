## Automated Network Request Management in ServiceNow

### Project Overview
This project is designed to automate network-related service requests using ServiceNow. It helps organizations manage network requests efficiently by reducing manual work and improving response time.

### Problem Statement
Manual handling of network requests leads to delays, errors, and lack of tracking. There is a need for an automated system to standardize request handling and approvals.

### Solution
The system allows users to submit network requests through a ServiceNow service catalog. Requester details are auto-filled, approvals are routed automatically, and tasks are created without manual intervention. Email notifications keep users informed about request status.

### Features
- Service Catalog-based network request
- Auto-populated user details
- Automated approval workflow
- Task creation and assignment
- Email notifications
- Request tracking and reporting

### Technologies Used
- ServiceNow
- Flow Designer / Workflow
- Business Rules
- Notifications

### Use Case
Suitable for IT service management teams to automate internal network-related requests such as connectivity, access, and device-related issues.
